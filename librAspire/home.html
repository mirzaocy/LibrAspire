Frontend Flow

Materi yang Sudah Diterapkan

Reusable component mindset

Kartu buku lokal dan kartu hasil pencarian eksternal memakai struktur tampilan yang seragam.

State dan interaksi

Status pinjam dan antre berubah tanpa memuat ulang halaman, lalu disinkronkan ke semua tampilan.

Routing berbasis URL

Setiap buku punya halaman detail sendiri lewat parameter URL untuk meniru konsep dynamic route.

Loading dan error state

Detail buku dan discovery API menampilkan kondisi memuat, gagal, atau kosong secara eksplisit.
